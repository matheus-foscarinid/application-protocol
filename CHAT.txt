# Iniciar conversa
HI IM <EMAIL>
# Validações e comportamento
 - Se for um Email já conectado ou um Email inválido ele nega a autenticação/conexão
 - Se tiver tudo OK retorna que deu certo e já retorna o histórico anterior de mensagens

# Enviar uma mensagem
SAY <mensagem>
 - Passa a mensagem que quer enviar, se for válida envia para todo o mundo
 - Precisa salvar em um histórico
 - Precisa ter um ID

# Consultar usuários
WHOS THERE
 - Retorna todos os usuários conectados (Emails)

# Editar uma mensagem
EDIT MSG <ID> <mensagem_nova>
 - Edita a mensagem passada

# Excluir uma mensagem
DELETE MSG <ID> <mensagem_nova>
 - Exclui a mensagem passada
